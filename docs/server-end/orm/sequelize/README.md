# SequelizeORM

![](images/sequelize-logo.png)

`Sequelize` 是适用于 `Oracle`、`Postgres`、`MySQL`、`MariaDB`、`SQLite` 和 `SQL Server`
等的现代数据库，支持 `TypeScript` 和 `Node.js`的`ORM`框架。具有可靠的事务支持、关系、急切和延迟加载、读取复制等功能。
