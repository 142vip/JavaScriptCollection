# 模板项目

`Express`框架提供了应用程序生成器，就是预先约定一些`Express`框架项目的开发规范，例如：`目录约束`、`路由约束`...

通过应用生成器工具 `express-generator`模块 可以快速创建一个应用的结构，可以在此基础上形成基于`Express`框架的模板项目
