## 题目描述

写一个bash脚本以统计一个文本文件nowcoder.txt 中每个单词出现的个数。

为了简单起见，你可以假设：
`nowcoder.txt`只包括小写字母和空格，每个单词只由小写字母组成，单词间由一个或多个空格字符分隔。

示例:
假设 `nowcoder.txt` 内容如下：

```text
welcome nowcoder
welcome to nowcoder
nowcoder
```

你的脚本应当输出（以词频升序排列）：

```text
to 1
welcome 2
nowcoder 3
```

说明:
不要担心个数相同的单词的排序问题，每个单词出现的个数都是唯一的。
