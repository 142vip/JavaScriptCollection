## 题目描述

编写一个shell脚本以输出一个文本文件`nowcoder.txt`中空行的行号（空行可能连续，从1开始输出）

示例:
假设 `nowcoder.txt` 内容如下

```text
a
b

c

d

e


f
```

你的脚本应当输出：

```text
3
5
7
9
10
```
