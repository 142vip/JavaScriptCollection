### 拉取镜像
```bash
$ docker pull rabbitmq
```
### 运行容器
15672是管理界面的端口，5672是服务的端口。这里顺便将管理系统的用户名和密码设置为admin admin

注意在网关kong中，设置ip

- 普通

```bash
$ docker run -d --name Myrabbitmq -e RABBITMQ_DEFAULT_USER=admin -e RABBITMQ_DEFAULT_PASS=admin -p 15672:15672 -p 5672:5672 rabbitmq:managemen
```
- 正式

```bash
$ docker run -d --name rabbitMQ-master -e RABBITMQ_DEFAULT_USER=admin -e RABBITMQ_DEFAULT_PASS=123456 -p 15672:15672 -p 5672:5672 --network=pga-net --ip=172.24.0.22 rabbitmq:management
```

### 设置ip

在docker创建容器的时候，可以利用`--network=kong-net`以及`--ip=127.0.0.2` 指定容器ip；同时也可以利用docker桥接网络的bridge操作去指定容器与桥接网络连接；
