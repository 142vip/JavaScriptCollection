## 题目描述

编写一个bash脚本以输出一个文本文件`nowcoder.txt`中第5行的内容。

示例:
假设 `nowcoder.txt` 内容如下：

```text
welcome
to
nowcoder
this
is
shell
code
```

你的脚本应当输出：

```text
is
```
