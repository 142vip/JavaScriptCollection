## IDEA

在官网下载官方依赖，选择符合系统的版本，然后安装

## uPic

简洁的 Mac 图床客户端 uPic

- 开源代码：<https://github.com/gee1k/uPic>
- 官网：<https://blog.svend.cc/upic/>

mac中安装：

```bash
## Homebrew安装
brew install bigwig-club/brew/upic --cask

## 出现错误，则更新brew
brew update-reset

```
